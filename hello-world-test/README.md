# Hello World Test Flow

This flow tests custom component infrastructure.

## Components Used
- HelloWorldEcho (from examples components)

## Creation Steps
1. Install components: `make langflow-import` (in multi-agent-platform)
2. Restart LangFlow: `make langflow-restart`
3. Create flow in LangFlow UI connecting: Chat Input -> Hello World Echo -> Chat Output
4. Export JSON to this directory
